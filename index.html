PDF → Sideways Text PDF (Binder-friendly)

Upload a PDF. This extracts the text and creates a new PDF where the text is rotated 90° so you can print it and use it in a top-opening binder.

Tip: If your output has weird spacing, try font size 9 or increase line spacing.
If the button stays gray after selecting a PDF, open DevTools (F12) → Console and look for red errors.
